# 概述仪表板


Adobe组合建模器中的概述仪表板允许您通过各种可配置的小组件概述您的协调数据和计划。

要访问概述，请选择 ![主页](../assets/icons/Home.svg) **[!UICONTROL Overview]** 从Adobe混合建模器界面的左边栏中。

从此处，您可以获取有关以下内容的更多详细信息：

* [协调的数据](harmonized-data.md)

* [计划](plans.md)
